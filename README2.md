## 🛠️ 第一步：环境一键部署（极速保姆级）

Windows 用户在配置量化环境时，最头疼的就是技术指标库 **`TA-Lib`** 的安装，因为在 Windows 下编译它需要配置 C++ 编译器，极易报错崩溃。

我们项目中自带的 **`install.bat`** 是一键部署神器，它会自动升级基础工具、自动下载安装为 Windows 编译好的 `TA-Lib` 库，并全自动安装好所有依赖。

请按照以下步骤进行 100% 安全的一键配置：

### 1. 确认 Python 版本
请确保您的电脑上安装了官方的 **Python 3.10**（推荐 64位）。

### 2. 创建虚拟环境
在项目根目录（`c:\AipythonProject\lianghua\vnpy`）下打开 PowerShell，运行以下命令创建独立的虚拟环境：
```powershell
# 创建名为 venv_quant 的独立虚拟环境
python -m venv venv_quant
```

### 3. 一键强绑定运行 `install.bat` 部署依赖
为了**绝对避免**将依赖库错装到全局电脑环境中，我们可以通过**传递参数**的方式，显式将虚拟环境的 Python 绑定给一键安装脚本。

在控制台中直接输入并运行以下命令（**无需提前激活虚拟环境，极度安全防错！**）：
```powershell
.\install.bat .\venv_quant\Scripts\python.exe
```
> **💡 此命令的底层原理与优势：**
> * 脚本会将第一个参数 `.\venv_quant\Scripts\python.exe` 识别为核心执行器。
> * 强制使用虚拟环境的 `pip` 升级并免编译下载安装 Windows 版 `ta_lib==0.6.4`。
> * **100% 确保所有库都安全隔离在 `venv_quant` 文件夹中**，绝不污染全局电脑环境，即使新手操作失误也绝对不会装错！

### 4. 激活虚拟环境并补全计算库
安装完成后，激活您的虚拟环境，并补充安装我们策略统计模块所需的风险指标计算库 `empyrical`：
```powershell
# 1. 激活刚刚装好所有依赖的虚拟环境 (命令行前会出现 (venv_quant) 字样)
.\venv_quant\Scripts\activate

# 2. 补装 empyrical 计算器
pip install empyrical
```

## 🔑 第二步：配置 TuShare 线上数据源

本系统需要连接 TuShare 线上接口自动补充历史数据。请确保您的 Token 已正确写入全局配置文件：

1. 打开全局配置文件路径：
   `C:\Users\<您的用户名>\.vntrader\vt_setting.json`  *(若没有该文件夹，可先运行一次 run.py 即可自动生成)*
2. 找到 `"datafeed"` 配置项，修改为：
   ```json
   "datafeed.name": "tushare",
   "datafeed.username": "您的TuShare账户注册手机号",
   "datafeed.password": "您的TuShare_API_Token"
   ```
   *(注：TuShare 积分需达到 120 积分以上才可顺利拉取 A 股日线历史数据。)*

## 🚀 第四步：如何使用批量全自动回测与增量数据系统

**批量脚本 `batch_research.py` 是本仓库的科研杀手锏，完美告别 GUI 的手动繁琐操作。**

### 1. 一键运行批量回测
在激活的虚拟环境中敲入：
```bash
python batch_research.py
```
**它会自动为您执行以下流水线：**
1. **智能缓存校验**：自动检查本地数据库中已有数据。
   * 若数据偏旧，仅向 TuShare 精准增量下载缺失的那几天“数据补丁”，绝不重复下载，0 浪费您的 API 积分，彻底免封禁！
   * 若数据已是最新，0 网络开销秒级跳过！
2. **极速后台回测**：依次为列表中的所有股票运行 `StockQuantStrategy` 8 年历史回测。
3. **输出 CSV 比对报表**：在项目根目录下生成一份精美的 **`batch_backtest_report.csv`** 表格。您可直接双击用 Excel 打开，多股收益、最大回撤、夏普比率横向比对一目了然！

### 2. 自由增删测试股票
打开 `batch_research.py`，定位至约 84 行 the 列表，按照格式增删股票：
```python

    ("000001", "SZSE"),  # 平安银行 (银行板块)
```

### 3. 灵活调节测试参数
如果您想快速测试不同的参数搭配，无需修改策略源码，直接修改 `batch_research.py` 约 80 行的大括号 `{}` 传参：

## 🧠 第五步：关于 A股极致多因子风控策略 (`StockQuantStrategy`)

该策略（位于 `strategies/stock_quant_strategy.py`）针对 A 股强波动与周期复苏特性进行了专业设计：
1. **三大因子抄底**：
   * **信号 1**：日线收盘连续 7 天下跌时，稳健买入 10% 仓位。
   * **信号 2**：股价进入过去 500 天最低 20% 分位时，买入 10% 仓位。
   * **信号 3**：股价创 3 日内新低时，轻仓买入 5% 仓位。
2. **独立止盈（不割肉逻辑）**：尊重 A 股跌深必反弹的周期律。每个子仓位单独记录入场价，一旦达到 `+3.5%` 精准止盈抛出。
3. **持仓硬性风控墙**：限制最多只能同时持有 5 个买入批次（最多使用 50% 资金，永远留存 50% 现金）。**此风控项将铜陵有色（000630）的最大回撤从 `-71.13%` 降至极为安全的 `-37.73%`，同时保持了 `102.06%` 的超额高回报，且在所有测试股票中爆仓率为 0**！
